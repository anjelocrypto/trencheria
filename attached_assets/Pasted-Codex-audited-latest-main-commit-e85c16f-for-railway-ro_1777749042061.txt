T001 — Add railway/road validation
Create a dev-only validator that checks:
- rail segments crossing rivers/lakes without a RAILWAY_BRIDGE

- railway too close to settlement collision footprints
- station platforms overlapping tracks, water, roads, or buildings
- trees/rocks/resources too close to railway
- railway/road intersections without a level crossing object
Print exact segment/station/bridge names when invalid.

- Line B around [-20,83] → [45,80]

Add or move RAILWAY_BRIDGES so every rail-water crossing has a bridge.

Add a railway ground-height helper, for example getRailGroundHeight(x,z), that uses railway bridge deck height when on a railway bridge, otherwise terrain height.
